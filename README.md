# zigwin32

A complete autogenerated set of Zig bindings for the Win32 API.

These bindings are generated by https://github.com/marlersoft/zigwin32gen

## Cold Bytes Games fork

This is a manually edited fork of zigwin32 that set the unicode_mode to ANSI and manually edit some Direct3D 12 functions to behave correctly, workaround for https://github.com/marlersoft/zigwin32/issues/6

Reason for using the ANSI version of Win32 API: [UTF-8 support in the Microsoft Game Development Kit](https://learn.microsoft.com/fr-ca/gaming/gdk/_content/gc/system/overviews/utf-8)
